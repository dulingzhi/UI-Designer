# 自定义参考线 - 快速上手指南

## 什么是参考线？

参考线是专业设计工具中的辅助功能，帮助设计师精确对齐UI元素。它们是临时的辅助线，不会出现在最终导出的代码中。

## 快速开始

### 1. 创建第一条参考线

```
步骤1: 确保标尺可见
      菜单: 查看 > 显示标尺

步骤2: 从标尺拖拽
      - 鼠标移到左侧垂直标尺 → 拖到画布 → 创建垂直参考线
      - 鼠标移到顶部水平标尺 → 拖到画布 → 创建水平参考线

步骤3: 释放鼠标
      参考线创建完成！
```

### 2. 使用参考线对齐

```
当你拖动UI元素时，它会自动吸附到附近的参考线：

拖动按钮 → 靠近参考线 → 自动吸附 → 精确对齐！
```

### 3. 调整参考线

```
鼠标悬停在参考线上 → 光标变化 → 拖拽移动 → 调整位置
```

### 4. 删除参考线

```
方式1: 双击参考线
方式2: 右键点击参考线
方式3: Ctrl+; 清除全部
```

## 使用场景

### 场景1: 多个按钮左对齐
```
1. 在按钮左边缘创建一条垂直参考线
2. 拖动其他按钮，它们会自动对齐到这条线
3. 完美的左对齐效果！
```

### 场景2: 网格布局
```
1. 创建多条等距的垂直参考线
2. 创建多条等距的水平参考线
3. UI元素按网格排列，整齐美观
```

### 场景3: 黄金分割
```
1. 在屏幕宽度的0.618位置创建垂直参考线
2. 在屏幕高度的0.618位置创建水平参考线
3. 在交叉点放置关键UI元素
```

## 提示和技巧

### 💡 提示1: 位置显示
拖动参考线时，会显示当前精确位置（像素值）

### 💡 提示2: 吸附范围
元素在距离参考线5像素内会自动吸附

### 💡 提示3: 颜色区分
- 蓝色 = 可编辑参考线
- 灰色 = 锁定参考线（不可编辑）

### 💡 提示4: 快捷操作
- `Ctrl+;` 快速清除所有参考线
- 双击比右键删除更快

## 常见问题

### Q: 参考线会导出到代码吗？
A: 不会。参考线仅用于设计辅助，不影响最终导出。

### Q: 如何创建精确位置的参考线？
A: 目前通过拖拽创建，拖拽时会显示位置。未来版本会支持输入精确值。

### Q: 参考线会保存吗？
A: 会！参考线随项目文件一起保存。

### Q: 为什么我的参考线是灰色的？
A: 灰色表示参考线被锁定，无法移动或删除。这是为了防止误操作。

### Q: 可以创建多少条参考线？
A: 理论上没有限制，但建议适量使用以保持画布清晰。

## 视觉示例

```
创建参考线前:

│  [按钮1]            │

│        [按钮3]      │
└─────────────────────┘

创建垂直参考线后:
┌─────────────────────┐
│ ┊ [按钮1]            │
│ ┊ [按钮2]            │
│ ┊ [按钮3]            │
└─────────────────────┘
        ↑
    参考线 (蓝色)

所有按钮左对齐!
```
